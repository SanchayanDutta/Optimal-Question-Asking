# OQA k-ary 25-object dataset

This package contains a 25-object multi-valued attribute dataset for Optimal Question Asking (OQA).
- Unique attribute vectors for all 25 objects.
- All attributes have at least two distinct values across the tier.
- Greedy information-gain decision tree expected depth: 2.40 queries (uniform prior).

- oqa_kary25_schema.json
- oqa_kary25_dataset.json
