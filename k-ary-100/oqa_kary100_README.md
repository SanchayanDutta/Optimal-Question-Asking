# OQA k-ary 100-object dataset

This package contains a 100-object multi-valued attribute dataset for OQA-style experiments.
- Unique attribute vectors for all 100 objects.
- All attributes have at least two distinct values across the tier.
- Greedy information-gain decision tree expected depth: 3.46 queries (uniform prior).

- oqa_kary100_schema.json
- oqa_kary100_dataset.json
- oqa_kary100_dataset.csv
- oqa_kary100_candidates.txt
- oqa_kary100_greedy_tree.json
- oqa_kary100_prompt.txt
